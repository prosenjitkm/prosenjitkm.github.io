# My Portfolio

Hello I am **Prosenjit**

This repository contains all the files and dependencies needed to craft my beautiful online portfolio.

## Technology Stack

- **Next.js** - React framework for production
- **TypeScript** - Type-safe JavaScript
- **Google Fonts** - Montserrat and Poppins fonts via CDN
- **CDN Icons** - Technology logos from jsDelivr (devicons)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## Build & Deploy

```bash
# Build static site
npm run build

# The output will be in the 'out' folder
# Deploy the 'out' folder contents to GitHub Pages
```

## Features

- ✅ Single page application
- ✅ Google Fonts optimization
- ✅ CDN-based technology logos (always up-to-date)
- ✅ Static site generation for GitHub Pages
- ✅ Fully responsive design
- ✅ TypeScript support
